Next: The Form & Preview Challenge

The next challenge is to build a two-state UI using Vanilla HTML, CSS, and JS. Create a form to collect user details. A "Preview" button should hide the form and display the entered data. In preview mode, the button should change to "Edit", which takes you back to the form. This task introduces managing UI "modes" or "views" with JavaScript.
